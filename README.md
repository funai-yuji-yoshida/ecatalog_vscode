# 📖 動的電子カタログ作成アプリ

FlipHTML5風のUI/UXを持つ、A4見開き形式の電子カタログ作成Streamlitアプリケーションです。

## ✨ 主な機能

### 1. FlipHTML5風の美しいUI
- A4見開き（左右2ページ）の本格的なカタログレイアウト
- ホバーエフェクトとスムーズなアニメーション
- グラデーション背景とシャドウ効果

### 2. 動的なブロック配置
- 1ページあたり1/2/4/6/8ブロックから選択可能
- 自動的にグリッドレイアウトが調整される
- 左右ページ独立して商品を配置

### 3. CSV動的バインディング
- 商品データをCSVから読み込み（Shift_JIS、CP932、UTF-8対応）
- 表示したいカラム（スペック項目）を自由に選択
- 各ブロックに任意の商品を割り当て可能
- データプレビュー機能

### 4. 画像管理（2パターン対応）
- **フォルダ指定**: ローカルフォルダから商品コードで自動マッチング
- **ファイルアップロード**: ブラウザから複数画像を一括アップロード
- PNG、JPG、JPEG形式に対応
- Base64エンコードで埋め込み

### 5. リアルタイムデザイン変更
- カラーピッカーでヘッダー色・枠線色を変更
- スライダーでフォントサイズ（10-28px）調整
- セル余白（5-25px）のカスタマイズ
- 全ブロックに即座に反映

### 6. ページ管理
- 複数の見開きページを管理
- 前後のページナビゲーション
- ページ番号の自動付与

### 7. 出力機能
- HTMLファイルとしてダウンロード
- 印刷最適化済み（@media print対応）
- 単一HTMLファイルで完結（スタンドアロン動作）

## 🚀 セットアップ

### 必要な環境
- Python 3.8以上
- pip（Pythonパッケージマネージャー）

### インストール手順

1. リポジトリをクローン
```bash
git clone https://github.com/funai-yuji-yoshida/ecatalog_vscode.git
cd ecatalog_vscode
```

2. 依存パッケージをインストール
```bash
pip install -r requirements.txt
```

3. アプリケーションを起動
```bash
streamlit run app.py
```

4. ブラウザで自動的に開かない場合は、以下のURLにアクセス
```
http://localhost:8501
```

## 📋 使い方

### ステップ1: データ読み込み
左サイドバーの「データ読み込み」セクションから、商品データのCSVファイルをアップロードします。

**CSVファイルの形式例:**
```csv
商品コード,③品番,④モデル名,⑦色,⑧入数
10400001,ABC-123,モデルA,レッド,10
10400002,DEF-456,モデルB,ブルー,20
```

### ステップ2: レイアウト設定
「レイアウト設定」セクションで、1ページあたりのブロック数（商品数）を選択します。
- 1ブロック: 大きく1商品を表示
- 2ブロック: 縦に2商品
- 4ブロック: 2×2グリッド
- 6ブロック: 3×2グリッド
- 8ブロック: 4×2グリッド

### ステップ3: 表示項目選択
「表示項目選択」セクションで:
1. カタログに表示したいカラム（スペック項目）を複数選択
2. 商品識別カラム（画像とマッチングするカラム）を選択

### ステップ4: 画像設定
「画像設定」セクションで、以下のいずれかの方法で画像を読み込みます:

**方法A: フォルダ指定**
```
例: C:/images/products
```
- 商品コード（例: 10400001.png）と一致するファイル名の画像を自動検索

**方法B: ファイルアップロード**
- ブラウザから複数の画像ファイルを選択してアップロード
- ファイル名（拡張子を除く）が商品コードとマッチング

### ステップ5: デザインカスタマイズ
「デザイン設定」セクションで:
- テーブルヘッダー色を選択
- 枠線の色を選択
- フォントサイズを調整
- セル余白を調整

### ステップ6: 商品配置
メイン画面の「商品配置設定」で:
1. 左ページ・右ページそれぞれのブロックを展開
2. 各ブロックに表示したい商品をドロップダウンから選択
3. プレビューがリアルタイムで更新される

### ステップ7: 出力
- 「HTMLをダウンロード」ボタンでHTMLファイルを保存
- 「印刷プレビュー」から印刷（Ctrl+P）

## 🐛 トラブルシューティング

### 問題: CSVが文字化けする
**解決策**: CSVファイルの文字コードを確認してください。Shift_JIS、CP932、UTF-8に対応していますが、他のエンコーディングの場合は変換が必要です。

### 問題: 画像が表示されない
**解決策**: 
1. 画像ファイル名が商品コードと完全に一致しているか確認
2. 対応形式（PNG、JPG、JPEG）であることを確認
3. ファイルパスに全角文字が含まれていないか確認

### 問題: レイアウトが崩れる
**解決策**: ブラウザのキャッシュをクリアして再読み込み（Ctrl+Shift+R）
